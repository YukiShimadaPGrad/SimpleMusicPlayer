# Name

シンプル超軽量ミュージックプレイヤ♬

## Overview

ローカル音楽ファイルを聞くための、無駄を省いた音楽再生Androidアプリ

- 広告なし
- インターネット通信なし
- 超軽量(アプリサイズ95KB)

## Requirement

Android 8.0 以上

## Features

- バックグラウンド再生
- リピート再生
- プレイリスト作成
- イコライザ
- バスブースト
- アルバムアート表示
